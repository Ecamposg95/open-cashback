# Roadmap

## Sprint 0

Base tecnica, FastAPI, PostgreSQL, Alembic, modelos, docs y Docker.

## Sprint 1

Auth completo, usuarios admin, roles y permisos multi-tenant.

## Sprint 2

Clientes, wallets y busqueda para POS.

## Sprint 3

Ventas, cashback engine, ledger EARNED e idempotencia persistente.

## Sprint 4

Redenciones, reversos, limites y antifraude basico.

## Sprint 5

Reportes y dashboard administrativo.

## Sprint 6

API keys, scopes, webhooks e integraciones POS/ERP.

## Sprint 7

Batch anchoring real con Ganache/testnet EVM.
